Frequently Asked Questions

Why does this show "Coming Soon"?
IoT features require physical sensors to be installed. The software is fully ready — we're just waiting for the hardware to be connected.
What hardware is needed?
HC-SR04 ultrasonic sensor for tank level, YF-S201 flow sensor for leak detection, ESP32 for Wi-Fi connectivity, and a relay module for pump control.
Will I need to update the software?
No. The API endpoints are already built. You just need to configure the hardware's IP address in DBConnection settings and the sensors will start feeding data.
Can users see IoT data too?
Yes. Once live, users will see their local tank level, water quality readings, and real-time usage on their personal dashboard.
