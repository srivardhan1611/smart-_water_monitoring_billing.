Setup Guide

Follow these steps in order to get the project running on your machine.

0

Before You Start — What You Need

Make sure these are installed on your computer:

1

Extract the Project

Unzip SBW_water_system_enhanced.zip anywhere on your computer. You will get a folder called sbw_enhanced.

sbw_enhanced/
  ├── src/com/water/       ← Java source files
  ├── WEB-INF/
      ├── web.xml             ← Servlet mappings
      └── lib/                ← MySQL JDBC driver
  ├── login.html             ← Login page
  ├── register.html          ← User registration (4-step)
  ├── dashboard.jsp          ← Admin portal
  ├── user_dashboard.jsp     ← User portal
  ├── iot_coming_soon.html   ← IoT hardware page

2

Set Up the MySQL Database

Open MySQL Workbench and connect to your local MySQL server. Then run the schema file:

Option A — Using MySQL Workbench (Easiest)

  1. Open MySQL Workbench → click your local connection
  2. Go to File → Open SQL Script
  3. Browse to sbw_enhanced/schema.sql and open it
  4. Click the ⚡ Execute button (or press Ctrl+Shift+Enter)
  5. You should see: Query OK messages and a result showing 4 users, 5 records

Option B — Using Command Line

-- Open your terminal / command prompt
mysql -u root -p
-- Enter your MySQL password when prompted, then:
source C:/path/to/sbw_enhanced/schema.sql;
✅ After running schema.sql, the database water_system will be created with 2 tables and 4 sample resident accounts ready to use.
3

Update Your MySQL Password in the Code

Open this file in any text editor (Notepad, VS Code, etc.):

sbw_enhanced/src/com/water/DBConnection.java

Find these two lines and change them to match your MySQL setup:

con = DriverManager.getConnection(
    "jdbc:mysql://localhost:3306/water_system?useSSL=false&serverTimezone=UTC",
    "root",        // ← Change this to YOUR MySQL username
    "manager"      // ← Change this to YOUR MySQL password
);

Common Examples:

Your MySQL SetupWhat to Write
Default install, no password"root", ""
Password is "root""root", "root"
Password is "admin123""root", "admin123"
Custom username "water_user""water_user", "yourpassword"
⚠️ This is the most common reason the project fails to connect. Double-check your MySQL password matches exactly — it is case-sensitive.
4

Compile the Java Files

Open a terminal / command prompt and navigate to the project folder. Then run the compile command.

On Windows:

cd C:\path\to\sbw_enhanced

javac -cp "WEB-INF\lib\mysql-connector-j-8.0.33.jar;C:\tomcat\lib\servlet-api.jar" ^
      -d WEB-INF\classes ^
      src\com\water\*.java

On Linux / macOS:

cd /path/to/sbw_enhanced

javac -cp "WEB-INF/lib/mysql-connector-j-8.0.33.jar:/opt/tomcat/lib/servlet-api.jar" \
      -d WEB-INF/classes \
      src/com/water/*.java
💡 Where is servlet-api.jar? It's inside your Tomcat folder at tomcat/lib/servlet-api.jar. Common paths:
Windows: C:\Program Files\Apache Software Foundation\Tomcat 9.0\lib\servlet-api.jar
Linux: /opt/tomcat/lib/servlet-api.jar

After this command, check that WEB-INF/classes/com/water/ now has .class files. If you see errors, re-check Step 3.

5

Deploy to Tomcat

Copy the entire sbw_enhanced folder into Tomcat's webapps directory:

-- Windows example:
Copy sbw_enhanced → C:\tomcat\webapps\sbw_enhanced

-- Linux example:
cp -r sbw_enhanced /opt/tomcat/webapps/sbw_enhanced

Then Start Tomcat:

-- Windows:
C:\tomcat\bin\startup.bat

-- Linux / macOS:
/opt/tomcat/bin/startup.sh
🚀 Wait about 5–10 seconds for Tomcat to fully start, then open your browser and go to:
http://localhost:8080/sbw_enhanced/

MySQL Database — Update Guide

Common database changes you will need to make.

A

Change a User's Password

Open MySQL Workbench, connect to your server, open a new query tab and run:

USE water_system;

-- Change password for a specific user:
UPDATE users
SET password = 'new_password_here'
WHERE username = 'saipranay';

-- Verify the change:
SELECT username, password FROM users WHERE username = 'saipranay';
💡 Replace 'saipranay' with any username from your users table, and 'new_password_here' with the new password.
B

Add a New Resident Manually

To add a user directly via SQL (instead of using the registration page):

USE water_system;

INSERT INTO users
    (first_name, last_name, email, username, phone, address, password, meter_no, role, is_active)
VALUES
    ('Ravi', 'Kumar', 'ravi@example.com',
     'ravikumar', '9876540000',
     'Flat 505, Block E, Hyderabad',
     'user123',
     'MTR1005',
     'USER', 1);

-- Verify:
SELECT id, username, email, meter_no FROM users;
⚠️ Make sure the email, username, and meter_no are unique — the table will reject duplicates.
C

View All Users & Their Details

USE water_system;

-- See all registered users:
SELECT id, first_name, last_name, username, email, phone, meter_no, created_at
FROM users
ORDER BY id DESC;

-- Count total users:
SELECT COUNT(*) AS total_users FROM users;

-- Find a specific user:
SELECT * FROM users WHERE username = 'saipranay';
D

View & Update Billing Records

USE water_system;

-- See all bills:
SELECT * FROM usage_records ORDER BY id DESC;

-- Bills for a specific user:
SELECT * FROM usage_records
WHERE customer_name = 'saipranay'
ORDER BY bill_date DESC;

-- Mark a bill as paid:
UPDATE usage_records
SET is_paid = 1
WHERE id = 3;

-- Total revenue collected:
SELECT
    COUNT(*)           AS total_bills,
    SUM(units)         AS total_litres,
    SUM(bill_amount)   AS total_revenue,
    SUM(CASE WHEN is_paid=1 THEN bill_amount ELSE 0 END) AS paid_amount,
    SUM(CASE WHEN is_paid=0 THEN bill_amount ELSE 0 END) AS pending_amount
FROM usage_records;
E

Deactivate or Delete a User

USE water_system;

-- SAFE: Deactivate (user can't login, data preserved):
UPDATE users SET is_active = 0 WHERE username = 'saipranay';

-- Reactivate a user:
UPDATE users SET is_active = 1 WHERE username = 'saipranay';

-- PERMANENT: Delete a user entirely (use with caution):
DELETE FROM users WHERE username = 'saipranay';
⚠️ Use UPDATE is_active = 0 (deactivate) instead of DELETE wherever possible. Deleting removes the user permanently and cannot be undone.
F

Reset / Clear All Data (Fresh Start)

USE water_system;

-- Clear only billing records (keep users):
TRUNCATE TABLE usage_records;

-- Clear only registered users (keep bills):
TRUNCATE TABLE users;

-- Clear EVERYTHING and start fresh:
SET FOREIGN_KEY_CHECKS = 0;
TRUNCATE TABLE usage_records;
TRUNCATE TABLE users;
SET FOREIGN_KEY_CHECKS = 1;

-- Verify tables are empty:
SELECT COUNT(*) FROM users;
SELECT COUNT(*) FROM usage_records;
⚠️ TRUNCATE is permanent. All data will be deleted. Only run this if you want a completely clean database.
